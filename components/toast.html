Toast 轻提示 ​

轻量的全局反馈提示，支持 success、error、info 三种类型，自动消失。

基础用法 ​

调用 showToast 函数触发提示。

ts
import { showToast } from "@/components/toast/toast";

showToast("操作成功", "success");
showToast("请求失败", "error");
showToast("提示信息", "info");

自定义时长 ​

第三个参数传入毫秒数，默认 2000ms。

ts
showToast("短暂提示", "info", 1000);
showToast("持久提示", "info", 5000);
